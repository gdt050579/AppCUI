Colors¶

AppCUI support 16 colors (foreground and background) for each displayed character. All colors are described using AppCUI::Graphics::Color enum:

enum class Color : unsigned char
{
    Black       = 0x00,
    DarkBlue    = 0x01,
    DarkGreen   = 0x02,
    Teal        = 0x03,
    DarkRed     = 0x04,
    Magenta     = 0x05,
    Olive       = 0x06,
    Silver      = 0x07,
    Gray        = 0x08,
    Blue        = 0x09,
    Green       = 0x0A,
    Aqua        = 0x0B,
    Red         = 0x0C,
    Pink        = 0x0D,
    Yellow      = 0x0E,
    White       = 0x0F,
    Transparent = 0x10,
};

Transparent value can be used to specify that the existing color should be used, instead of replacing it with the new one.
